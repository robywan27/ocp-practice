> fixed set of constants providing type-safe checking

> constructors are implicitly private (no public or protected constructors allowed)

## Kinds
1. simple: contains only a list of values
2. complex: can contain (static and/or final) fields, (static and/or final or abstract) methods and constructors

## Special syntax
| What                                             | Description               | Syntax                            |
|--------------------------------------------------|---------------------------|-----------------------------------|
| constructor calls (when defining list of values) | don't use the new keyword | enumValue(constructorArgs)        |
| instance method                                  |                           | [enumName.]enumValue.methodName() |

## Methods
| Method name     |
|-----------------|
| T[] values()    |
| valueOf(String) |
| name()          |
| ordinal()       |
