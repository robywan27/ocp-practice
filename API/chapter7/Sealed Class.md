> class that restricts which classes may extend it

## Definition
| Class/Interface        | Modifier          | Description                            |
|------------------------|-------------------|----------------------------------------|
| Parent class/interface | <b>sealed</b>     |                                        |
|                        | <b>permits</b>    | list of extending/implementing classes |
| Subclass/interface     | <b>final</b>      | can't be further extended              |
|                        | <b>sealed</b>     | same rules as parent sealed class      |
|                        | <b>non-sealed</b> | can be extended by any (unknown) class |

## Special syntax
| What                         | Description                                                  | Syntax                                        |
|------------------------------|--------------------------------------------------------------|-----------------------------------------------|
| permits in nested subclasses | must be prefixed with the name of the enclosing sealed class | permits EnclosingSealedClassName.SubclassName |

## Rules
| MUST                     |                                                                                                                         | CANNOT                                      |                                                                                  | CAN                                                       |                                                                                                |
|--------------------------|-------------------------------------------------------------------------------------------------------------------------|---------------------------------------------|----------------------------------------------------------------------------------|-----------------------------------------------------------|------------------------------------------------------------------------------------------------|
| a sealed class/interface | - be declared in the same package as its direct subclasses or<br/>- in the same named module even in different packages |                                             |                                                                                  |                                                           |                                                                                                |
|                          | must list all direct subclasses in its permits clause (except when conditions below apply)                              |                                             |                                                                                  |                                                           |                                                                                                | 
| a subclass/interface     | each extend the sealed class                                                                                            |                                             |                                                                                  |                                                           |                                                                                                |
|                          |                                                                                                                         | a sealed class/interface                    | list indirect (more than one level of inheritance) classes in its permits clause |                                                           |                                                                                                |
|                          |                                                                                                                         | an interface that extends another interface | be final (interfaces are implicitly abstract), but only sealed or non-sealed     |                                                           |                                                                                                |
|                          |                                                                                                                         |                                             |                                                                                  | a sealed class/interface                                  | omit permits if:<br/>- all subclasses are defined in the same file<br/>- subclasses are nested |
|                          |                                                                                                                         |                                             |                                                                                  |                                                           | be declared abstract                                                                           |
|                          |                                                                                                                         |                                             |                                                                                  | a sealed interface                                        | be implemented by classes and be extended by interfaces (listed after permits)                 |
|                          |                                                                                                                         |                                             |                                                                                  | a sealed class/interface in a switch statement/expression | omit the default branch when all permitted subclasses are covered and parent class is abstract |
