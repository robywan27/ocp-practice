> Strings are immutable

## Initialization
| Method                     | Description                                                                                           |
|----------------------------|-------------------------------------------------------------------------------------------------------|
| var s = new String("abc"); | regular reference type initialization (note: this explicitly creates a new object in the string pool) |
| var s = "abc";             | string literal initialization                                                                         |
| var block = """<br>abc"""; | text block initialization                                                                             |

## Equality
> s1.equals(s2); // checks content of strings

> String Pool - a special memory region where string literals are stored; when a string literal is created, the JVM checks if an identical string already exists in the pool and reuses it instead of creating a new one

## String concatenation
> \+ operator (remember rules when mixing with other types)
